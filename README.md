# History
历史是什么，过去的一分一秒都是历史，但是能被记录下的历史是较少的。我们为什么要学习历史，无论是学习语文(为什么要写这篇文章)、数学(公式解决了什么问题)，学习到具体的篇幅都会先提到一个背景，当了解背景以后再切入学习，这些背景就是历史，历史就记录了这些。当我们把历史的发展串联起来会发现历史的产生是有一定的规律的，历史不是凭空产生的。学习历史，了解历史，尊重历史。
